# 五大逼出 AI 極致潛力的技巧

## 技巧 1：給明確的角色 + 專業框架

❌ **普通問法：**
```
幫我寫一個腳本
```

✅ **極致問法：**
```
你是 Netflix 的資深編劇，專精 15 秒短影片。

框架：
- Hook（0-2秒）：製造懸念
- 轉折（3-10秒）：情緒堆疊
- 結尾（11-15秒）：意想不到

用這個框架，寫一個關於 [主題] 的腳本。
```

---

## 技巧 2：給範例（Few-shot Learning）

AI 特別擅長從範例學習：

✅ **極致問法：**
```
參考這些成功案例的結構：

範例 1：
標題：『這個方法讓我一週省下 10 小時』
結構：[描述]
效果：10 萬觀看

範例 2：
標題：『專家看到這個都驚訝了』
結構：[描述]
效果：5 萬觀看

現在用同樣的模式，幫我寫一個關於 [主題] 的。
```

---

## 技巧 3：要求結構化輸出

AI 對結構化輸出特別穩定：

✅ **極致問法：**
```
輸出格式必須是：

{
  "標題": "...",
  "摘要": "一句話",
  "重點": ["1", "2", "3"],
  "下一步行動": "..."
}

不要加任何解釋，只要這個格式。
```

---

## 技巧 4：分步驟但給自主權

✅ **極致問法：**
```
任務：[描述任務]

步驟：
1. 先分析 [什麼]
2. 設計 [什麼]
3. 產出 [什麼]
4. 自我檢查品質

每個步驟完成後，用你的專業判斷優化。
如果發現更好的方法，直接採用。
```

---

## 技巧 5：挑戰性標準 + 自我迭代

✅ **極致問法：**
```
寫 5 個版本，然後：
1. 自己評分（1-10）
2. 分析每個版本的優缺點
3. 結合所有優點，寫出第 6 個最強版本
4. 解釋為什麼第 6 個最強
```

---

## 模型選擇建議

| 任務類型 | 用快速模型 | 用深度模型 |
|---------|----------|-----------|
| 寫程式碼 | ✅ 極致 | 過度 |
| 結構化內容 | ✅ 極致 | 過度 |
| 批量生成 | ✅ 極致 | 太慢 |
| 策略規劃 | 可以 | ✅ 更深 |
| 創意發想 | 可以 | ✅ 更野 |
| 深度分析 | 可以 | ✅ 更透 |

## 萬用極致 Prompt 模板

```
# 角色
你是 [領域] 的頂尖專家。
專精：[具體專長 1]、[具體專長 2]、[具體專長 3]

# 任務
[具體描述要做什麼]

# 框架
每個輸出必須包含：
- [元素 1]
- [元素 2]
- [元素 3]

# 輸出格式
[指定格式，如 JSON、表格、條列]

# 標準
- [品質標準 1]
- [品質標準 2]
- 自我評估後只輸出最強版本
```
